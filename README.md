# 📖 Overview

AI Calculator is a modern web-based calculator that goes beyond simple calculations.

It combines traditional calculator functionality with an AI-inspired math assistant capable of understanding natural language queries, solving mathematical expressions, supporting voice input, and providing step-by-step solutions.

This project was built to improve JavaScript skills while experimenting with intelligent user interactions and modern UI design.
